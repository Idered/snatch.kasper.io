Snatch

Beautiful clipboard manager for Windows. Keep your history under control, literally.

Download for Windows View source on GitHub

Features at a glance

Unlimited history

Snatch will keep every copied element until you decide to clean the history.

Quick search

No matter how long history you have, you can search it in just a fraction o second.

Pinned entries

You can pin any entry. Pinned entries will stay in place when list is cleared.

What lies ahead

Fuzzy search

Find entries that approximately match search string.

Images support

Keep history of copied images.

Settings

One setting doesn't fit all. Anyone should be able to change any part of Snatch.
